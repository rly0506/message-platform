## 主线(一句话,焊死)

**手机负责标记好奇心,电脑负责消化好奇心;省力早报喂事实,硬核台增理解——两模式不互相污染。**

为什么是入口先行:U2 对照/事件图已建成,但产品是高意图工作台,缺低意图入口(见记忆 `casual-open-painpoint`)。不解决"够得着",理解层建再多也闲置。人类三条当日痛点(不想开电脑/日报没重点/认知外无法提问)全指向入口层。

## 双模式定义(张力2 决议)

| | 省力早报 | 硬核拓展 |
|---|---|---|
| 场景 | 手机/通勤/累 | 电脑/有精力 |
| 判据 | 低摩擦,3 分钟能停 | 有益的不适 |
| 内容 | **事实摘要**(谁/何时/可核查主张)+ 今日一个领域 | 多源对照/类比/事件图/戳盲区 |
| 载体 | 邮件推送(纯省力终点站)+ 头版 | 事件分析台 |
| 桥 | 「回头深挖」标记 → **深挖队列** | 队列消化 + 头版深挖入口(A)+ 事件详情内联(C) |

**深挖队列**(张力2 补充决议):承认省力和硬核天然发生在不同时间不同设备,不硬逼当场跳转。手机标记 → 队列缓冲 → 电脑消化。

## 已定决议记录(免得后轮翻聊天记录)

1. **入口 = A+C**:A 头版卡片深挖按钮(新增,省力→硬核的桥);C 事件详情内联对照(现状保留)。不做独立"理解"tab(B 被否:埋太深,低意图到不了)。
2. **张力1(因果树)决议**:底层永远是神经元网(证据关系,参照虐杀原形意识网形态);"根源/因果"作为**可召唤的假设叠加层**(=透镜3/F3),默认关、灰虚线+「假设」角标、标推断、给矛盾证据。形态给用户,语义分层。
3. **张力3(本地 ML)决议**:聚类/PCA/随机森林存为工具箱**不立项**;等 U1 类比落地后若简单相似度不够,拿聚类打擂台再决定引入 scikit-learn。不因"学过"而做。
4. **早报喂事实不喂结论**:摘要 = 可核查事实,不是替用户思考的叙事(阿德勒红线,见记忆 `information-vs-understanding`)。
5. **每日领域轮询 V1 = 早报里的「今日一个领域」卡**:教怎么想这个领域(多源怎么框/像历史什么/未决问题),不喂结论。它是认知地图的引导程序(破"认知外无法提问"死循环)。
6. **源扩展并行**:政军外交国际组织官方一手源,GPT 门禁间隙做,实测新鲜度纪律不破。

## Claude 轨(frontend/ 独占)

### Phase 1 · 双模式入口 V1a(本轮重心,纯前端零后端依赖)
1. **覆盖差异点回证据**:`EventContrast.vue` gap 行用 `evidence_article_ids`(后端已精确化)映射 `sources[].articles` url,渲染可点链接。
2. **深挖队列 V1a(localStorage)**:新 composable `useDigQueue.ts`(薄存储接口,V1b 换 marks API 只换适配层)。标记:头版种子卡 + 事件详情「回头深挖」;消化:分析台顶部「待深挖 N 件」卡带 → 点击切 topic+选中事件+展开对照。
3. **头版深挖入口(A)**:头版卡片按钮 → `trackTopic` + 定位事件 + 展开对照台。
4. **Deep-link**:`App.vue` 解析 `?topic=&event=&view=contrast`(无 router),为邮件深链铺路。
5. 每批 build + 全量 e2e 绿,独立提交(人类终审)。

### Phase 2 · U1 类比卡带(等 GPT 后端契约,shell-first 可提前)
「相似先例」卡带:相似依据(basis 逐项)+ **差异提醒必显**(类比不预言)。消费 `/events/{id}/analogues`。

### Phase 3 · F3 假设层占位(填充)
事件图「假设层」开关(默认关):灰/虚/「假设」角标 + 未开启态文案。只做占位,完整 U3(LLM 假设生成)单独立项。

## GPT 轨(backend/ 独占,经 TO_CODEX 派单,每步 8 步流程)

1. **U1 类比后端**(侦察已 ACCEPT,按其自提 TDD 顺序)。Claude 审查细化:
   - `differences[]` 证据化具名(「时间相差 N 月」「对方 top 实体含 X 本事件无」),不用模板话;
   - 候选池全库线性扫可接受,注明规模天花板,超限截断留痕;
   - `score_label` 档位阈值写死+测试;
   - 认:V1 不写 EventRelation(防事件图重建误删)、无 Event 行话题 degraded 不 sync(保持只读)。
2. **dig_later 标记落库**(小):CognitionMark label 集扩 `dig_later`,供队列 V1b 跨设备,复用 marks GET/PUT。
3. **源扩展批次**(并行填充):政军外交国际组织官方源,feeds.json + 逐源实测新鲜度,僵尸标 zombie 不硬编,延续 0B 守卫。
4. **早报升级后端半**(Phase 3 末,先方案后写码):每链接事实摘要(诚实注明全文天花板:基于 RSS 标题+摘要)、邮件深链、「今日一个领域」卡。

## 红线(沿用 + 新增两条)

沿用:无 LLM 也能跑/降级留痕/证据可核查/禁词纪律(导致/根因/证明/因果)/不新增重依赖(no vector DB)/不写真实库/不合并 master 不推 origin(人类定)/GPT 只碰 backend、Claude 只碰 frontend/幽灵文件不提交/改符号前 GitNexus impact。
新增:**早报喂事实不喂结论**;**两模式不互相污染**(省力早报不塞成瘾钩子,硬核台不为好看牺牲对照诚实)。
