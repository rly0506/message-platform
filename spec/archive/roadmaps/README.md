## Archive Rules

1. Do not execute a file here without checking `spec/roadmap-ledger.md` and the current code.
2. Do not delete or rewrite historical conclusions; add a new roadmap or ledger entry when the project changes direction.
3. Keep stable RM IDs even when files move. New executable roadmaps receive their ID in the ledger before work starts.
